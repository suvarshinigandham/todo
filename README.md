# 📝 TODO App (JavaScript Project)

## 📌 Description

This project is a simple **TODO (Task Management) Web Application** built using **HTML, CSS, and JavaScript**.
It allows users to add, mark as completed, and delete tasks in an interactive and user-friendly interface.

This project demonstrates basic **DOM manipulation** and **event handling** in JavaScript.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed (line-through)
* ❌ Delete tasks
* ⚡ Instant updates without page reload
* 🎨 Clean and responsive UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🧠 How It Works

* User enters a task in the input field
* Clicking **Add** creates a new task item
* Clicking on the task marks it as completed
* Clicking **Delete/X** removes the task

---

## 📸 Output

* Input field to enter tasks
* Add button
* Task list displaying:

  * Task name
  * Complete option
  * Delete button
